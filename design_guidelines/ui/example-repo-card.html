Page guideline · Example repo card

Send the reader to the example.

An example repo card lifts a runnable folder on GitHub out of the prose. Octocat disc + monospaced path + one-line description, with a small "star us" pill on the right as a soft kicker. The whole card is a single link to the example folder — one click target, the star pill is decorative. Use it once per post, right where you'd otherwise paste a long github.com/... link.

01 — Anatomy

Octocat · path · description · star pill.

Octocat sits in a maroon-ink disc on the left. The path is monospaced and reads as a real folder path. The description supports <strong> to flag the star ask. The right-hand star us pill is decorative — the whole card is one anchor, so clicking anywhere takes the reader to the example.

Example · GitHub examples/csv_to_kafka → Clone it, point it at a Kafka broker, and watch a folder of CSVs flow into a topic. If it saves you an afternoon, drop a ⭐ on the repo — that's how more connectors get prioritized. star us
02 — Usage

Where it goes, where it doesn't.

Reach for it where you'd otherwise paste a long github.com/cocoindex-io/... URL into prose. One per post is right; two starts to feel beggy.

Do

Place it where the reader is about to run something.

Right after the prose says "the full example lives at…". The card replaces that sentence — don't leave both.

Do

Bold the star ask in the description.

<strong> wraps the "drop a ⭐" part. The bold pulls the eye back to the star pill, which is the actual click.

Don't

Don't use it as the post's closing CTA.

End-of-post star asks belong in the maroon `.cta` strip, not here. This card is a mid-article hand-off; the strip is the goodbye.

Don't

Don't stack two of them.

One example per post. If a post points to multiple repos, link them inline — the card's job is to spotlight a single runnable thing.

03 — API

Astro component props.

Wrapper component lives at src/components/ExampleRepoCard.astro. Import from MDX. description accepts HTML so authors can use <strong> and emoji to flag the star ask.

PropTypeNotes
href string · required Direct link to the example folder, e.g. github.com/cocoindex-io/cocoindex/tree/main/examples/csv_to_kafka.
path string · required The mono label rendered inside <code>. Match the repo path so the card reads as a real file path.
description string · optional HTML-permitted. One sentence; bold the star ask with <strong>. Add an emoji ⭐ if it helps the eye.
eyebrow string · default "Example · GitHub" The kicker. Override for non-GitHub examples ("Example · GitLab" etc).
starLabel string · default "star us" Pill label. Lowercase mono looks right; keep it short (≤ 8 chars).
